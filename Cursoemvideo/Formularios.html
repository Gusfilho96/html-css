Formulários:

Utilizando <form> podemos criar um formulário. Usamos o Action para informar para onde os arquivos vão ser enviados, no caso do exemplo a baixo é PHP

Input:

Existem vários tipos de input, é importante que exista uma ligação entre o input e o label. Isso serve para o navegador saber a quem aquele input está ligado.

As ligações dependem do tipo de linguagem que você ira utilizar. HTML e PHP funcionam melhor com Name, já JavaScript funciona melhor com ID.

Color

Esse tipo de input premite que o usuario selecione alguma cor, podemos usar value para iniciar com alguma cor.

Range

Esse metodo funciona para presquisa de satisfação por exemplo, podemos colocar valor min ou max e o value para o nivel inicial.

File

Esse metodo permite enviar algum arquivo, ao utilizar esse recurso é importante utilizar o metodo Post, caso contrario o arquivo não será enviado.

Checkbox & Radio

Checkbox

O input de tipo checkbox serve para uma marcação simples, um exemplo seria a marcação de esportes que a pessoa pratica. Utilizar o label nesse caso server tanto para o navegador saber a quem aquele input está ligado, e servira também para que o usuario utilize a palavra para marcar a checkbox.

Checkbox

Radio

Já o radio pode ser utilizado quando deve-se marcar apenas uma opção, um exemplo seria a seleção do sexo. Nesse caso o name deve ser o mesmo em ambos dentro daquele fieldset, já label pode ser utilizado normalmente. Outra coisa importante é o value, um simples value="M" ou "F" ajudara na hora do submit.

Radio

Get vs Post

O método Get serve para informações não sigilosas até 3000Bites, esse metodo serve, também para que possamos enviar uma URL que contenha aquela informação, um exemplo legal seria as pesquisas do Google

Já o método Post, apesar de ser um pouco mais seguro, ele ainda permite que as informações sejam acessadas.

Importante!!! Estude HTTPs para que as informações fiquem realmente seguras.

Agrupar

Utilizando <fieldset> funciona para agrupar os campos contidos nessa tag, isso pode ser usado em conjunto com a tag <legend> dentro, possibilitando que todos os dados pessoais, por exemplo fiquem dentro de um campo.

Atributos

Required = Campo obrigatório.
Min/Max Lenght = Tamanho maximo e/ou minimo.
Placeholder = Escrita temporária dentro da caixa.
Autocomplete = Preenchimento automático, funciona com informações salvas no browser.
Min/Max = Valor minimo e/ou maximo.
