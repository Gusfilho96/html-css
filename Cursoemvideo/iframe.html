Todos os sites aceitam Iframe?

Não, um dos problemas do Iframe é que muitos sites não aceitam ser "compartilhados" via Iframes. No caso do YouTube, é a forma que ele deixa você hospedar vídeos no seu site.

Tamanho Padão:

Padrão: 300x150. Podemos adicionar um outro tamanho pelo próprio Iframe ou pelo CSS (CSS sempre soberano nesse caso)

Navegação

  • Home
  • Pagina 1
  • Pagina 2
  • Pagina 3

Podemos Utilizar Iframe e adicionar um Nome nos parâmetros em conjunto com target="nome" para abrir a pagina no iframe.

Segurança

O maior problema de usar Iframe em páginas fora do site, é o fato de você ficar vulnerável a qualquer ataque no naquele site. Fora isso, tem o fato de que o site pode alterar o conteudo e trazer problemas para o seu site.

Detalhes:

É uma boa sempre colocar uma mensagem ou link dentro do Iframe, dessa forma, caso o navegador não consiga exibir o conteúdo os usuários vão conseguir acessar o site.

Scrolling="yes" ou "no" esse parâmetro não funciona em todos os navegadores.
