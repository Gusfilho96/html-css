Estilos

Em Linha

<p style="background-color: yellow;">Texto do parágrafo</p>

Não recomendado para varias modificações.

Local

<style>
h2 { background-color: black; }
</style>

Basta colocar style dentro da parte do head, isso modifica itens

Interno

<link rel="stylesheet" href="style.css" />

Criano da mesma forma na parte do head, porém ele utiliza um arquivo CSS para poder modificiar todas as paginas com esse link

Id / #

Utilizando o id e #, no caso para identificação no html e css respectivamente, você pode colocar um estilo personalizado naquele item especifico.

Class / .

A mesma ideia se aplica aqui. utilizando class para juntar varios itens em um id apenas. importante lembrar de identificar class como um . no css.

Pseudo-classes / :

Configura um estado de um elemento. Pode usar para adicionar uma cor ao passar o mouse, ou para mostrar algo escondido ao passar por cima.

Pseudo-elemento / ::

Um exemplo de uso, é utilizar para adicionar um label para todo link por exemplo.

Representações de Cores.

Código Hexadeimal

#a52a2a

Código RGB

rgb(165, 42, 42)

Código HSL

hsl(0, 59%, 41%)

OBS: também podemos muudar a transparencia.
