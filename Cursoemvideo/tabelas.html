Hierarquias

É importante manter sempre a hierarquias, já que mesmo se colocado no topo um item que não segue a hierarquia, vai surgir em um local diferente. Abaixo a ordem:

Caption = <caption>
Table = <table>
Row = <tr>
Header = <th>
Data = <td>

OBS: O Fechamento dessas tags é opcional.

Anatomia

Podemos utilizar algumas outras tags para organizar melhor as tabelas de forma semantica. São elas: <thead>, <tbody> e <tfoot>.

Zebrado

Utiliza-se a pseudoclass: :nth-child(even), :nth-child(odd), :nth-child(n2) e :nth-child(2n-1)

Selecionar Coluna ou Linha

Usando <colgroup>, <rowgroup> e dentro <col>, <row> podemos criar classes e/ou ids para nossas colunas, inclusive usar span para modificar multiplas colunas ou linhas
