Tabelas

  • Hierarquias É importante manter sempre a hierarquias, já que mesmo se colocado no topo um item que não segue a hierarquia, vai surgir em um local diferente.
  • <thead>, <tbody> e <tfoot> Para tabelas maiores é preferivel separar as partes da tabela.
  • <caption> título da tabela
  • <table> A tabela
  • <tr> A linha da tabela
  • <th> Tableheader
  • <td> Data
  • Zebrado Utilizando pseudoclass: podemos modificar todo filho impar ou par.
    • :nth-child(even) ou :nth-child(n2) para linhas pares
    • :nth-child(odd) ou :nth-child(2n-1) para linhas impares.
  • Seletor de coluna ou linha
    • Usando <colgroup> ou <rowgroup> e dentro <col> ou <row> podemos criar classes e/ou ids para linhas ou colunas, inclusive utilizar span para modificar multiplas linhas ou colunas.
  • Criando <div> odemos usar overflowpara criar uma barra de rolagem horizontal e/ou vertical na tabela.
