Forma Semântica

  • <!--...--> ou Alt+Shift+A Cria um comentário.
  • <h1> até <h6> Define um comentário e suas hierarquias.
  • <span> Define uma parte do texto.
  • <br> Quebra a linha.
  • &+lt e &+gt Mostra < e > respectivamente.
  • <mark> Utilizado para marcar o texto.
  • <code> Utilizado para digitar códigos no site.
  • <pre> Define um texto pré formato
  • <a> Define um hyperlink
    • rel="external" Indica que o link aponta para um site externo.
    • rel="next" Indica que o link leva a próxima página.
    • rel="prev" Indica que o link leva a página anterior.
    • rel="_blank Faz o link abrir em uma nova aba ou janela.
    • rel="_self" Faz o link abrir na mesma aba.
  • <q> Citação simples, utilizada após o texto.
  • <blockquote> e atributo cite="link" para cotações mais completas e com link direto.
  • <img> Utilizado para inserir imagem interna ou de alguma fonte externa no site. Redimencionar imagem com CSS não altera o tamanho do arquivo.
  • <audio> Utilizado para inserir áudios no site.
    • Source + src="..." e type="..." para definir a origem do audio, caminho e formato.
    • Atributos preload e controls
    • Parâmetros do preload: auto, metadata e none
  • <video> Inserir um vídeo de forma interna, bem parecida com a tag: audio. OBS: Usar de forma pontual, vídeos internos consomem bastante.
  • <Iframe> O iframe basicamente adiciona uma janela dentro do site, pode servir para adicionar uma tabela, vídeo, site externo ou site interno. Apesar de ser possivel alterar o height do iframe dentro da própria tag, é preferivel utilizar css, já que esse é priorizado. Outra coisa que pose ser feita é usar o atributo scrollin="no"/"yes/"auto"
  • <ol> Lista do tipo ordenada, ótima para listas númeradas.
  • <ul> Lista do tipo não ordenada.
  • <li> Item da lista.
  • <dl> Lista de definição.
  • <dt> Item a ser definido.
  • <dd> Definição do item.
    • type e start para modificar o simbolo da lista e onde ira iniciar. Esse segundo atributo serve mais para listas ordenadas.
  • <form> Funciona de forma parecida das listas, basicamente precisa de outras tags e atributos
