Coisas Básicas

Hierarquias:

Utiliza-se h1 a h6 para ditar quem é sub-diretório de quem

Seleção de paragrafo

Posso usar a tag span para fazer algo com uma parte do paragrafo.

Semantica vs não Semantica:

Basicamente Semantica traz um significado, vai ser utilizado no Html. Já não semantica, se trata de estilos, no caso Css.

Qubra de linhas:

Tag: br

Como mostrar < e >

Tag: & + lt e gt

Símbolos especiais:

Tag: & + reg, copy, trade, euro, pound, cent, dollar, yen, real, etc.

Emojis:

Tag: &#x + Codigo do emoji.

Favicon:

Bom site para utilizar: favicon.io
Em head, usar Tag:link:favicon

Negrito:

Tag: strong

Itálico:

Tag: em

Letra muidas:

Tag: small

Excluido:

Informa que deve ser lido, mas não considerado. Tag: del

Inserido:

Texto adicionado depois. Tag: ins

Sobrescrito:

Muito util nesse tipo de situação: X2+3. Tag: sup

Subscrito:

Util para esse tipo de situação: H2O. Tag: sub

Abreviações:

Basicamente passar o mouse por cima e exibir significado. Tag: abbr

Comentário no code:

Para Html usar: < + ! + - + -

Marcações:

Tag: mark

Códigos:

Para deixar algum codigo escrito no site, basta usar a Tag: code
Caso precise, usar antes de code a Tag: pre

Atalhos:

Envelopar algo: Ctrl + Shift + P

Atalho de caminho: Crtl + Espaço

Criar item: Crtl + Click na palavra

Organizar codigos: Tab ou Shift + Tab
